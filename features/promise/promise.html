Actividad 4: Promises

¿Qué es una Promise?

"¡Prometo un resultado!"

Las promesas de JavaScript se crearon para que el uso de JavaScript asincrónico sea más fácil. Un objeto Promise representa la finalización o el fracaso de una operación asincrónica. Una Promesa puede estar en uno de tres estados exclusivos:
PENDIENTE, RECHAZADO, CUMPLIDO.
